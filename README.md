# AI-Based RF Interference Detection Assistant

An AI-powered assistant developed using **IBM watsonx Orchestrate**, **IBM Granite**, and **RAG** to analyze RF signal data, identify interference sources, and recommend mitigation strategies.

## Features

* RF Signal Analysis
* RF Interference Detection
* Mitigation Recommendations
* Knowledge-Based Responses (RAG)
* Interactive AI Assistant

## Technologies Used

* IBM watsonx Orchestrate
* IBM Granite Model
* IBM Cloud
* Retrieval-Augmented Generation (RAG)
* Knowledge Base
